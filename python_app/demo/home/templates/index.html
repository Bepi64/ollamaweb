Présentation du projet

Ce projet consiste à créer un site web de chat visuel en utilisant Django.
L'objectif est de permettre aux utilisateurs d'échanger avec des modèles d'intelligence artificielle Ollama directement depuis une interface web simple et intuitive.

  • Django : Framework web Python pour la gestion du backend et des vues.
  • Ollama : Modèles d'IA pour générer des réponses aux messages des utilisateurs.
  • HTML, CSS, JavaScript : Technologies utilisées pour l'interface utilisateur.

Ce projet est idéal pour découvrir l'intégration de l'IA dans des applications web modernes.
